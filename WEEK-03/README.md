# IS 224 Web Development Lab Week 3

This project is a practical implementation of HTML and CSS concepts based on the IS 224 Web Development Lab instructions.

# Objective

The objective of this lab is to:

- Apply HTML webpage structure
- Use CSS styling methods
- Demonstrate different CSS selectors
- Create a simple personal webpage

# Features Implemented

# HTML Structure

The webpage contains:

- Header section
- Navigation menu with anchor links
- About section with paragraph and image
- Skills section using unordered list
- Daily routine section using ordered list

# CSS Styling Methods

The project demonstrates all three CSS application methods:

1. External CSS
2. Internal CSS
3. Inline CSS

# CSS Selectors Used

The following selectors are implemented:

- Element selector
- Class selector
- ID selector
- Group selector
- Descendant selector
- Pseudo-class selector (`:hover`)
