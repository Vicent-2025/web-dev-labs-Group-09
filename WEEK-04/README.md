# CSS CONTROL LAYOUT

This project demonstrates fundamental CSS layout control techniques using:

- Box Model
- Positioning
- Float and Clear

The implementation follows the requirements provided in the lab document.

# Project Objective

The objective of this lab is to understand how layout control works in CSS by manipulating:

- Margin
- Padding
- Border
- Positioning
- Float
- Clear

The project intentionally avoids modern layout systems such as:

- Flexbox
- CSS Grid

This helps build a strong understanding of traditional CSS layout behavior.

# Features Implemented

# 1. Box Model

Three content boxes were created to demonstrate:

- Padding
- Margin
- Border

# Observations

- Padding creates internal spacing
- Margin separates boxes
- Border wraps content and padding

# 2. Positioning

The project demonstrates different positioning methods:

# Relative Position

Used on content boxes to create a positioning reference.

# Absolute Position

Labels such as:

- NEW
- US
- CONTACT

are positioned at the top-right corner of boxes.

# Fixed Position

The header remains visible while scrolling.

# Sticky Position

The navigation bar sticks to the top during scrolling.

# 3. Float and Clear

A two-column layout was created using:

- `float: left`
- `float: right`

# Layout Structure

- Main Content
- Sidebar
- Footer

# Clear Property

The footer uses:

css
clear: both;

This prevents overlap caused by floated elements.
